# Vue Pair Programming Test - Eduardo Juarez solution

This is the solution for the activities from the pair programming test

## What I implemented

### Add reset button and logic to Counter component

### Implemented a Todo List with:
- Default items
- Add item functionality
- Delete item functionality
- Text filtering
- “No results” state
- Reset functionality that restores defaults and clears the filter

### Added localStorage persistence for the todo list state & Counter

### Added ItemFilter component

### Added test files

## Setup Instructions

1. **Clone the repository**:
   ```
   git clone <repository-url>
   cd vue-pair-programming-test
   ```

2. **Install dependencies**:
   ```
   npm install
   ```

3. **Run the application**:
   ```
   npm run dev
   ```

4. **Run test**:
   ```
   npm test
   ```
